h1. set-proxy

h2. Purpose

Sets the proxy Grails should use when communicating over the internet such as with the "install-plugin":install-plugin.html  command

h2. Examples

bc.
grails set-proxy 

h2. Description

Usage: 
@grails set-proxy@

Arguments:

* *http.proxyHost* - The host name of the proxy server
* *http.proxyPort* - The port of the proxy server
* *http.proxyUserName* - The username of the proxy server
* *http.proxyPassword* - The password of the proxy server
